# Exercise Naming Convention Inconsistencies

## Summary
Found several categories of naming inconsistencies across 437 exercises.

---

## 1. HYPHEN vs SPACE INCONSISTENCIES

### Reverse-Grip vs Reverse Grip
- **With hyphen (2 exercises):**
  - Reverse-Grip Dumbbell Press
  - Reverse-Grip Preacher Curl
- **With space (3 exercises):**
  - Reverse Grip Bench Press
  - Reverse Grip Lat Pulldown
  - Reverse Grip Tricep Pushdown

### Smith-Machine vs Smith Machine
- **With hyphen (1 exercise):**
  - Smith-Machine Guillotine Press
- **With space (4 exercises):**
  - Smith Machine Bench Press
  - Smith Machine Incline Bench Press
  - Smith Machine Shrug
  - Smith Machine Squat

### Trap-Bar vs Trap Bar
- **With hyphen (1 exercise):**
  - Trap-Bar Jump Squat
- **With space (2 exercises):**
  - Trap Bar Hold
  - Trap Bar Deadlift

### Behind-the-Back vs Behind the Back
- **With hyphen (1 exercise):**
  - Behind-the-Back Barbell Shrug
- **With space (1 exercise):**
  - Behind the Back Deadlift

### Close-Grip (consistent - all use hyphen)
- Close-Grip Bench Press
- Close-Grip Incline Bench Press
- Close-Grip Lat Pulldown
- Close-Grip Push-Up
- Close-Grip (appears consistently with hyphen)

### Wide-Grip (consistent - all use hyphen)
- Wide-Grip Bench Press
- Wide-Grip Push-Up
- Wide-Grip (appears consistently with hyphen)

### One-Arm vs Single-Arm
- **"One-Arm" (1 exercise):**
  - One-Arm Lat Pulldown
- **"Single-Arm" (7 exercises):**
  - Single-Arm Dumbbell Bench Press
  - Single-Arm Landmine Press
  - Single-Arm Cable Row
  - Single-Arm Cable Bicep Curl
  - Single-Arm Cable Triceps Extension
  - Single-Arm Dead Hang
  - Single-Arm Dumbbell Preacher Curl

**Note:** These mean the same thing but use different terms. Should be standardized to one term.

### Single-Leg vs Single Leg
- **With hyphen:**
  - Single-Leg Glute Bridge
  - Single-Leg Hip Thrust
  - Single-Leg Deadlift
  - Single-Leg Dumbbell Deadlift
  - Single-Leg Romanian Deadlift
  - Single-Leg Seated Calf Raise
  - Single-Leg Press
  - Single-Leg Squat
- **With space:**
  - Single Leg Lat Pulldown (wait, this should be "One-Arm" - checking...)

### Plate-Loaded (consistent - all use hyphen)
- Plate-Loaded Chest Press
- Plate-Loaded Decline Chest Press
- Plate-Loaded Incline Chest Press
- Plate-Loaded Shoulder Press
- Plate-Loaded Standing Calf Raise
- Plate-Loaded High Row
- Plate-Loaded Low Row

## 2. CAPITALIZATION INCONSISTENCIES

### Lowercase After Hyphen
- **Behind-the-Back Barbell Shrug** - "the" is lowercase (should probably be "Behind-The-Back" or "Behind the Back")

### Other Capitalization Patterns
- Most exercises capitalize the first letter after a hyphen (e.g., "Close-Grip", "Wide-Grip")
- "Behind-the-Back" is the exception with lowercase "the"

---

## 3. ABBREVIATION INCONSISTENCIES

### EZ Bar vs EZ-Bar
- **With hyphen:**
  - EZ-Bar Preacher Curl
  - EZ-Bar Reverse Curl
- **With space:**
  - EZ Bar Curl

### Tricep vs Triceps
- **"Tricep" (singular - 4 exercises):**
  - Dumbbell Tricep Kickback
  - Machine Tricep Extension
  - Reverse Grip Tricep Pushdown
  - Tricep Pushdown With Bar
  - Tricep Rope Pushdown
- **"Triceps" (plural - 6 exercises):**
  - Cable Triceps Kickback
  - Lying Barbell Triceps Extension
  - Lying Dumbbell Triceps Extension
  - Overhead Cable Triceps Extension
  - Seated Dumbbell Triceps Extension
  - Standing Barbell Triceps Extension

---

## 4. PARENTHESES USAGE

Only 2 exercises use parentheses:
- Bench Dips (Feet Elevated)
- I-Y-T Raise (on Bench)

These are consistent in their usage pattern.

---

## 5. "AND" vs "&" USAGE

All exercises use "and" (not "&"):
- Clean and Jerk
- Clean and Press
- Dumbbell Clean and Press

This is consistent.

---

## 6. OTHER PATTERNS

### "Pull-Up" vs "Pull Up"
- All use hyphen: Pull-Up, Chin-Up, etc. (consistent)

### "Sit-Up" vs "Sit Up"
- All use hyphen: Sit-Up, Decline Sit-Up, etc. (consistent)

### "Push-Up" vs "Push Up"
- All use hyphen: Push-Up, Close-Grip Push-Up, etc. (consistent)

### "Leg Raise" vs "Leg-Raise"
- Mixed: "Hanging Leg Raise" (space) vs "Elbow-Supported Leg Raise" (space)
- Actually, all seem to use space for "Leg Raise" (consistent)

---

## RECOMMENDATIONS

1. **Standardize Reverse-Grip/Reverse Grip** - Choose one format (recommend hyphen: "Reverse-Grip")
2. **Standardize Smith-Machine/Smith Machine** - Choose one format (recommend space: "Smith Machine" since majority uses space)
3. **Standardize Trap-Bar/Trap Bar** - Choose one format (recommend space: "Trap Bar" since majority uses space)
4. **Standardize Behind-the-Back/Behind the Back** - Choose one format (recommend space: "Behind the Back" for consistency with "Behind the Neck Press")
5. **Standardize EZ Bar/EZ-Bar** - Choose one format (recommend hyphen: "EZ-Bar" for consistency)
6. **Standardize Tricep/Triceps** - Choose one form (recommend "Triceps" as it's the correct anatomical term)
7. **Standardize One-Arm/Single-Arm** - Choose one term (recommend "Single-Arm" since it's used in 7 exercises vs 1)
8. **Fix "Behind-the-Back"** - Either capitalize "The" or use space format
